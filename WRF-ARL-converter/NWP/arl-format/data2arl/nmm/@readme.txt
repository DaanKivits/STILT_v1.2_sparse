./data2arl/nmm  			Last Revised: 17 Nov 2003 
----------------------------------------------------------------------
This directory contains a grib decoder for NOAA's non-hydrostatic
mesoscale model (NMM) for grib data files in the native horizontal
grid format (4 or 8 km resolution) and on the model's sigma surfaces.
The sigma conversion to ARL format is only valid from the surface 
to 400 hPa.
