./data2arl/afwa 			Last Revised: 17 Nov 2003 
----------------------------------------------------------------------
This directory contains a grib decoder for AFWA's MM5 model grid 
output in grib format for various resolution and projections. These
MM5 output fields are not in the native MM5 format, but have already
been interpolated to some other projection, prior to their conversion
to GRIB.

	afwa2arl - 40 km NCEP/ETA format on the AWIPS 212 grid
	pNA45    - 45 km resolution Lambert Conformal grid
	pNA15    - 15 km resolution Lambert Conformal grid
	pNA05    -  5 km resolution Lambert Conformal grid
